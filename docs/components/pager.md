##  Pager - 页码
### 示例
<ClientOnly>
<pager-demos></pager-demos>
</ClientOnly>

### 代码
```HTML
<g-pager :totalPage="16" :currentPage.sync="currentPage"></g-pager>
```




### Attributes
| 参数 | 说明 | 类型 | 默认值 |
| ------ | ------ | ------ | ------ |
| totalPage | 总条数（必填） | Number |   无|
| currentPage | 当前页数（必填） | Number | 无 |
