## Icon-图标
### 示例
<ClientOnly>
<icon-demos></icon-demos>
</ClientOnly>


### 代码
```HTML
<div>
    <g-icon name="success"></g-icon>
    <g-icon name="error"></g-icon>
    <g-icon name="warning"></g-icon>
    <g-icon name="info"></g-icon>
</div>

```

### Attributes
| 参数 | 说明 | 类型 | 默认值 |
| ------ | ------ | ------ | ------ |
| name | icon名称 | String |   无|
